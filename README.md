# Global Exchange Works (국제교류처 업무 통합 관리 시스템)

국제교류과의 업무, 시간, 책임, 범위를 명확히 연결하여 업무 효율성을 높이는 통합 관리 시스템입니다.

## 1. 주요 기능

### 📊 대시보드 (Dashboard)
- **실시간 현황**: 진행 중, 대기 중, 마감 임박 업무 수치를 한눈에 확인
- **마감 임박 업무**: 7일 이내 마감되는 중요 업무 Top 5 리스트
- **진행률 차트**: 전체 업무 상태 비율 시각화

### 🗓️ 업무 달력 (New!)
- **FullCalendar 연동**: 월간/주간/일간 뷰 지원
- **마감일 관리**: 마감 기한이 설정된 업무를 달력에서 직관적으로 확인
- **상태별 색상**: 완료(초록), 긴급(빨강), 일반(파랑)으로 구분

### 👥 직원 및 업무 분장 (New!)
- **담당자 카드**: 총 9명의 직원 프로필 및 담당 업무(R&R) 상세 조회
- **명확한 책임 소재**: 누가 어떤 업무를 담당하는지 리스트업 되어 있어 업무 배정 시 혼선 방지

### 📝 업무 관리 (Task Management)
- **전체 리스트 뷰**: 담당자별, 상태별 필터링 및 검색 기능 제공
- **칸반 보드**: '대기 - 진행 - 완료' 상태별 업무 흐름 시각화
- **업무 등록/수정**:
    - **담당자 지정**: 등록된 직원 DB에서 담당자 선택
    - **마감일 설정**: 정확한 데드라인 관리
    - **우선순위**: High, Medium, Low 설정

### 📈 통계 및 분석 (Analytics)
- **직원별 업무량**: 차트를 통해 특정 직원에게 업무가 편중되었는지 확인 가능

## 2. 포함된 데이터 (최신 업데이트)
요청하신 최신 업무 분장 데이터를 반영하여 직원 9명의 정보가 등록되어 있습니다.
- **등록 인원**: 이진중(과장), 이혜진, 신지효, 오상훈, 김세원, 심정아, 박희경, 장예지, 나예은
- (김희정, 조민주 제외됨)

## 3. 기술 스택
- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Libraries**: FullCalendar, Chart.js, Font Awesome
- **Backend**: RESTful Table API
